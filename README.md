## STEP I: Generate your SSH Key on your local machine

- Generate Key:

```bash
ssh-keygen -t ed25519 -f ~/.ssh/[desired_keyname]
```

- List generated key:

```bash
ls ~/.ssh
```

- View public key of the SSH

```bash
cat ~/.ssh/[keyname]
```
